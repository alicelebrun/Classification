# Classification
Classification d'objets utilisant scikit-learn. On développe un algorithme de classification permettant de prédire l'état (efficace/inefficace) d'un objet décrit par trois paramètres (nombre d’heures d’utilisation
quotidienne, consommation électrique et âge de l’objet en mois) en utilisant différentes techniques (classification logistique, réseau de neurone)

Les algorithmes sont implémentés dans le notebook https://github.com/alicelebrun/Classification/blob/1ad366c33aa85eb0404cda550982479ad49177ff/ENT306_Alice_LEBRUN_sujet_2.ipynb
